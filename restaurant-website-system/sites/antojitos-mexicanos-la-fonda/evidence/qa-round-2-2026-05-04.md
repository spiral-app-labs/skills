# Antojitos Mexicanos La Fonda — QA Round 2 (2026-05-04)

## Final verdict

Pass after fixes. QA Round 2 focused on founder sellability: whether any page still looked internal, over-absolute, cramped on mobile, or unclear on conversion. The initial pass found near-blockers; all were fixed and rechecked.

## What is world-class already

- **Identity:** The site now clearly belongs to La Fonda: Veracruz-style antojitos, tacos, sope Veracruzano, empanadas, huaraches, mole/atole, public review proof, clean carryout, and Crystal Lake details are visible.
- **Conversion:** Calls and directions are clear on desktop and mobile; sticky mobile CTA is present; `/contact` keeps phone, address, hours, directions, and public profile links easy to reach.
- **Accuracy:** The copy avoids fake ordering/provider/pricing/seating claims and frames uncertain details as call-to-confirm.
- **Design quality:** `pepper-01` remains bright and casual without collapsing into generic Mexican-site copy. The proof strip, dish cards, concierge, and footer feel coherent.
- **Mobile polish:** Mobile has no horizontal overflow, the footer no longer creates a blank post-footer band, and the once-cramped path cards now stack readably.

## QA findings fixed in this round

1. **Internal/preview-facing language**
   - Removed/softened internal-feeling footer labels and copy (`Menu proof`, `Evidence`, `Preview build by Spiral`).
   - Footer now uses public-facing labels: `Menu highlights`, `Public profiles`, and a source-review note.

2. **Absolute no-site language**
   - Changed `No official site found` / absolute no-site presentation to safer `Website not verified` language.
   - Proof strip now says `Real demand. Website not verified.`

3. **Cramped mobile path cards**
   - Changed the visit/path card grid from two columns on small mobile to one-column stacking with a practical minimum height.
   - Reframed the section from audit language (`A Cleaner Path Than Directory Hopping`) to public-facing copy: `Everything You Need Before You Go`.

4. **Conversion hierarchy / internal CTA risk**
   - QA checked for `Visit mobile site now` and found no remaining visible instance after fixes.
   - Hero remains call/directions-first.
