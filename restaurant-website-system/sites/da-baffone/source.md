## First-Party Sources Used

- Current WordPress pages: home, about, contact, dinner menu, dessert menu, wine/libations, press posts.
- Media library: 2024 slider photos, historic interior photos, food/drink images, Best of the Fox badges.
- Public review surfaces: Tripadvisor, Yelp, Facebook, Restaurant Guru/Google aggregate.

## Implementation Positioning

This fork keeps Gusto's warm dark canvas, italic display serif, testimonial overlay, hover image treatment, and editorial menu layout, but changes the conversion strategy to phone-first reservations. The scrollable homepage is the default pitch because it is more familiar to restaurant owners; the compact cinematic version tests appetite for the more aesthetic Gusto interpretation.
