# Bistro Wasabi

Scrollable qitchen-01-inspired fork for Bistro Wasabi.

## Direction

- Dark refined sushi/fusion visual spine from `templates/qitchen-01`
- Scrollable, conversion-aware homepage
- Shared brand site with two location entry points
- Lake in the Hills Tock, Toast carry-out, Toast gift cards, and Uber Eats preserved
- Phone, email, published hours, and Hoffman Estates location preserved
- No formal omakase claim
- Final seller materials in `pitch-doc.md` and `pitch-battle-card.md`

## Run

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
