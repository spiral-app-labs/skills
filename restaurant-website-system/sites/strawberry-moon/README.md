# Strawberry Moon

First preview fork of `velvet-shaker-01`, adjusted for Strawberry Moon's real public identity: martinis, wine, nibbles, decadent desserts, live music, and a cozy two-level Wauconda lounge with first-come seating.

## Run

```bash
npm install
npm run build
```

If the host cannot reach npm, reuse a compatible local `node_modules` tree from another Next 14 site in this repo before running `npm run build`.

## Notes

- Official evidence lives alongside the app in `audit.md`, `checklist.md`, `checklist.json`, `scrapes/`, and `screenshots/`.
- This preview intentionally avoids fake reservations, fake ordering, fake hours, and fake menu prices.
- See `source.md` for the factual inputs and route decisions.
