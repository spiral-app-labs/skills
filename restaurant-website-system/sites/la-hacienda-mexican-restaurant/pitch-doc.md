## One-line pitch

La Hacienda already has the local demand, longtime regulars, and menu favorites. The preview turns a Canva-menu-only web path into a diner-ready homepage with clear menu, call, directions, order, hours, and safe concierge handoffs.

## 60-second owner read

La Hacienda is not a restaurant that needs fake polish or an invented story. The public evidence already gives us a practical sales angle: 4.3 stars from 530 Google reviews at capture, repeat mentions of tacos, burritos, tortas, chimichangas, fajitas, salsa, horchata, fast pickup, friendly service, and longtime regulars.

The leak is that the current web path behaves like a menu deck instead of a real first impression. Guests can find the menu if they already know where to look, but the owned experience does not quickly answer the questions mobile diners have before they choose: What should I order? Are you open? Where are you? Can I call or order pickup?

The preview fixes that by putting those decisions in the first screen and keeping the whole site grounded in what La Hacienda already is: a warm East Dundee neighborhood Mexican restaurant with familiar favorites and fast next steps.

## Why this is worth showing the owner

- **There is real public proof to build from.** Google showed 4.3 stars from 530 reviews during capture, and the Highest-sort packet includes 30 written reviews.
- **The current site path undersells the restaurant.** A Canva menu deck carries menu information, but it does not behave like a conversion homepage for hungry mobile guests.
- **The menu has obvious hooks.** Tacos, burritos, tortas, specials, chimichangas, fajitas, horchata, salsa, and pickup-friendly items are easy to frame.
- **The fix is practical, not abstract branding.** The preview focuses on View Menu, Call, Order, Map, hours, address, and safe handoffs.
- **The AI concierge is intentionally conservative.** It answers verified basics and routes allergy, price, and time-sensitive questions back to the phone.

## Current-site pain points

1. **Menu deck instead of a homepage.**
   The current Canva path can show menu content, but it does not immediately sell La Hacienda as a restaurant, surface trust, or organize action paths.

2. **Mobile guests have to assemble the basics themselves.**
   A first-time diner should not need to piece together phone, address, hours, menu, and order path from directories.

3. **Public reviews are doing work the owned web path should support.**
   Reviews already point to food, portions, service, salsa, and repeat loyalty. The preview translates those themes into the site structure.

4. **Order/call/directions are not treated as the main product.**
   For a casual neighborhood Mexican restaurant, the revenue path is often fast decisions: call, browse, order, or drive over.

## What the preview fixes

### 1. A real first impression

The homepage now opens as a La Hacienda page, not a file viewer. It leads with East Dundee Mexican favorites and the practical actions guests need before dinner.

### 2. Mobile conversion

The mobile experience now has a four-action strip for Call, Menu, Order, and Map. Header copy was tightened to avoid awkward truncation, and 390px overflow checks found no horizontal offenders on home, menu, and contact.

### 3. Menu scan path

The menu page gives guests fast anchors around tacos, burritos, tortas, specials, and bigger plates, while still handing off to the full public menu link.

### 4. Trust without fake claims

The preview uses safe themes from the audit and review packet: regular favorites, fast pickup, friendly service, longtime regulars, and familiar menu items. It does not invent chef stories, awards, reservations, catering, delivery providers, or sourcing claims.

### 5. Safe AI concierge

The `Ask La Hacienda` concierge can answer menu recommendations, hours, pickup, address, and phone basics. It sends allergy, ingredient, price, and time-sensitive pickup questions to the restaurant phone.
