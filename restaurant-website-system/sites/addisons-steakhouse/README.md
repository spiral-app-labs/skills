# Addison's Steakhouse

Warmed-down `1776-redesign-01` fork for Addison's Steakhouse in McHenry, IL.

## Direction

- Warm neighborhood steakhouse, not destination fine dining
- Phone-first reservation path made mobile-tappable
- Real menu sections and prices pulled from the current site
- From-scratch kitchen story centered around Chef Michael Borlek and owner voice
- Bar room, martinis, specials, patio, and occasion use cases surfaced without cluttering global nav
- Audit and outreach notes preserved in `audit.md` and `outreach.md`

## Run

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
