# Placeholder photos

These photos are downloaded from the original Framer Qitchen template (Framer's CDN: `framerusercontent.com`) so the catalog reference recreation looks faithful even after Framer rotates / removes the source URLs.

**These are PLACEHOLDERS for the catalog. Forks must replace them.**

When forking this template for a real restaurant:

1. Replace every photo in this folder with the restaurant's own photos.
2. Photos must match qitchen's locked aesthetic — see `research/template-audits/qitchen-01.md` §12.4:
   - Dark backgrounds (concrete, wood, near-black surroundings)
   - Warm lighting (golden, amber, candle-warm)
   - Low-key exposure
   - No bright daylight, no high-saturation colors
3. Update `content.example.ts` (or your fork's `content.ts`) image paths if you change filenames.

If the restaurant doesn't have on-brand photography yet, the audit's §12.5 is explicit: that's a routing failure — recommend a different template instead, OR commission a photographer before building.
