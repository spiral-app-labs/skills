# agent-foundation-files

Use when creating or revising role foundation packs for agents.

## Required Files per Agent
- `IDENTITY.md`
- `SOUL.md`
- `AGENTS.md`

## Must Include
### IDENTITY
- role mandate
- accountability scope
- collaboration interfaces
- decision heuristics
- success definition

### SOUL
- core truths/doctrine
- quality standards
- anti-patterns
- communication style
- operating posture

### AGENTS
- mission
- owns / does-not-own
- standard workflow
- quality gates
- escalation triggers
- done message contract

## Quality Tests
- distinguishable personality in first 3 lines
- clear authority boundaries (who decides what)
- objective escalation thresholds (time/risk/ambiguity)
- cold-start executable by a new agent

## Mirror Requirement
If founder asks, mirror full content to Notion for review/edit.
